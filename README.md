## 📖 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ApexBuildersGMBH.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
